Продукт предназначен для любых конфигураций 1С, соответствубщих требованиям:
* Режим запуска управляемое приложение
* Режим совместимости 8.3.10 и выше
* Функциональность БСП поддерживается с версии 2.4 и выше
* Языки конфигурации: русский, английский
